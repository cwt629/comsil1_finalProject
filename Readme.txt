20161255 장원태 - Sorting Game

프로젝트 실행 환경은 Visual Studio, Debug - Win32입니다.

아래 언급하는 파일들을 "소스" 폴더에 따로 분류하여 넣어놨습니다.
1. main.cpp, ofApp.cpp, ofApp.h을 src 디렉토리에 저장해주십시오.
2. BestRecord.txt(최고기록 저장 파일)를 bin 디렉토리에 저장해주십시오. bin/data 디렉토리에 저장하시면 안 됩니다.
3. verdana.ttf(폰트 파일)를 bin/data 디렉토리에 저장해주십시오.

프로그램을 실행하게 되면,
콘솔에는 조작법과 최고기록을 보여줍니다.
어느 때든 게임을 종료하려면 x키를 누르시면 됩니다.

게임 시작 화면에서 s키를 누르시면 게임이 시작됩니다.
이후 3초의 카운트다운이 시작되며, 그 후에 30초간 게임을 진행합니다.

좌, 상, 우 방향키를 입력하여 각 방향으로 블록을 같은 색의 타일로 보냅니다.
블록을 잘못 보내게 되면, 우측 상단의 라이프가 감소합니다.
좌측 상단에는 시간이 지속적으로 흐르는 것을 확인할 수 있습니다.

라이프가 모두 소진되거나 시간이 모두 지나가면 게임이 종료되며, 
x키를 눌러 프로그램을 종료하면 그 때까지 분류한 블록의 개수를 콘솔을 통해 알려줍니다.
